Agent Features
Available to every agent after login
🔐
Login Live
Sign in with your existing back-office email and password. No new account or reset needed.

Dashboard Live
Your production at a glance: volume closed, deals this year, and recruiting network with residual income. Pulls live from transaction history.

My Profile Live
Edit your own name, email, phone, and up to 8 social media links. Changes save directly to the agent directory.

DotLoop Transactions Pending
Connect your DotLoop account to view all your loops by status, edit closing details and commissions, and preview documents — without leaving AgentEdge. Waiting on OAuth credentials.
Admin & Back Office
Retention admins, MC leaders, and above
📋
Agent Roster Live
Full directory of all INNOVATE agents. Sortable, live search, contact info, and social links in one view. Market center shown for every agent.
🛡️
Roles & Permissions Live
Six access levels: super admin · retention admin · recruiter · broker in charge · MC leader · agent. Each role sees only the tools it needs.
➕
Agent Onboarding Live
Intake form that collects all contact details and creates the new agent's record in the system in one step. Retention admin and above.
🔍
Login as Agent Coming soon
Super admins can view AgentEdge as any agent for troubleshooting and onboarding support. A persistent banner shows whose account you're viewing.
🏡
Open House Tool Coming soon
Schedule and manage open houses with MLS listing integration. Porting from the existing agent-analytics tool with expanded data.
⚙️
Cap Wheel Coming soon
Visual cap progress tracker showing how much each agent has paid and how much remains before they cap. Requires Darwin (AccountTECH) integration.
